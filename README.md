# ⛅ Weather Dashboard

A beautiful desktop weather application built with Python Tkinter. Get real-time weather information, view forecasts, and save your favorite locations.

## ✨ Features

- Real-time weather data display
- Search any city worldwide
- Save default city for next launch
- Beautiful modern UI design
- Current date and time display
- Weather icons based on conditions
- Detailed weather metrics (wind, humidity, UV, sunrise/sunset)
- Day/Night mode indicator
- Refresh button for updates

## 📋 Requirements

- Python 3.6 or higher
- tkinter (included with Python)
- Custom `project.py` with `weather_api` class

## 🚀 Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/weather-dashboard.git

# Run the app
